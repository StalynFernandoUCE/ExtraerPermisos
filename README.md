# Extraer Permisos de APKs (Android) con Python

<h2>¿Cómo ejecutar?</h2>
Copie sus APKs en la carpeta Malware y ejecute el script en la terminal.

```bash
python3 ExtractorAIO.py
```
El script hará el resto.
>Esto puede demorar varios minutos según el tamaño y la cantidad de sus archivos APK.


<h2>¿Cómo utilizar los datos generados?</h2>
Los datos generados estarán en formato .csv y se pueden analizar con la ayuda de muchas bibliotecas o módulos precompilados.
> Se sugiere el módulo pandas en python

La primera columna contiene el nombre de la aplicación respectiva y la última columna "CLASE" contiene información si la aplicación es de una familia benigna o de malware del conjunto de entrenamiento.

<h2>Si se presenta algun problema al momento de ejecutar siga las siguientes recomendaciones</h2>

a)	Instalar Java SE 11</br>
Revisar la versión instalada en la máquina Linux</br>
java –version</br>
En caso que la versión sea menor a java SE 11 ejecute los siguientes pasos en orden, en caso de que aparezca un mensaje de error omita y continue con el siguiente comando.</br> 
Sudo rm -rf /var/cache/oracle-jdk11-installer-local</br>
sudo rm /var/lib/dpkg/info/oracle*</br>
sudo apt purge oracle-java11-installer*</br>
sudo rm /etc/apt/sources.list.d/*java*</br>
sudo apt autoremove</br>
java –version</br>
sudo dpkg --list | grep -i jre</br>
sudo apt remove default-jre</br>
sudo apt autoremove default-jre</br>
java –version</br>
sudo apt-get autoremove openjdk-7-jre</br>
Se procede a instalar la versión Java SE 11</br>
	sudo apt-get update</br>
	sudo apt install openjdk-11-jre-headless</br>
java-version</br>
sudo apt install openjdk-11-jre</br>
En caso de necesitar puede instalar Libreoffice</br>
	sudo apt install libreoffice</br>
 
b)	Nos ubicamos dentro de la carpeta Android-Permission-Extraction<Nombre_test> y ejecutamos el comando</br>
python3 ExtractorAIO.py</br>
      En caso de que se presente el siguiente mensaje de error</br>
	Picked up _JAVA_OPTIONS: -Dawt.useSystemAAFontSettings=on -Dswing.aatext=true</br>
      Ejecute los comandos</br>
	_SILENT_JAVA_OPTIONS="$_JAVA_OPTIONS"</br>
unset _JAVA_OPTIONS</br>
alias java='java "$_SILENT_JAVA_OPTIONS"'</br> 
      Vuelva al ejecutar.</br>
